## Bus Ticket Reservation System — Django Full-Stack Project:

This is a full-stack Bus Ticket Reservation System built using Django, Python, SQL, HTML, CSS, and JavaScript.
The system allows passengers to search buses, check seat availability, and book tickets, while the admin manages all buses, schedules, and bookings using Django ORM.

Project Overview:

This project supports two main roles:

1. User (Passenger):
   
	•	Creates an account and logs in

	•	Searches for buses by route, date, and time

	•	Checks real-time seat availability

	•	Books tickets and views booking history.


3. Admin:
	•	Adds and manages buses, routes, and schedules

	•	Updates seat availability

	•	Views all passenger bookings

	•	Handles backend logic entirely through Django ORM

	•	Can modify, delete, and update database entries.

Tech Stack:

Backend:

	•	Python
	•	Django Framework
	•	Django ORM
	•	SQLite

Frontend:

	•	HTML
	•	CSS
	•	Django Templates

Features Implemented:

	•	User authentication (login, register)
	•	Bus search with date & route filters
	•	Real-time seat availability
	•	Ticket booking functionality
	•	Booking history for users
	•	Admin dashboard for bus, route & schedule management
	•	Fully working backend using Django ORM
	•	Frontend built with HTML + CSS.
